# 🍽️ Recipe App

## 👩‍💻 Author

Ginger Korporaal

---

This is a React application built as part of my Full Stack Development course at Winc Academy.

## 📌 Features

- Browse a list of recipes
- Search recipes by name or health labels
- Filter recipes (Vegan, Vegetarian, Pescatarian)
- View detailed recipe information
- See ingredients and nutritional values
- Responsive design with Chakra UI
- Light/Dark mode toggle

## 🚀 Getting Started

1. Clone the repository
2. Install dependencies:
   npm install
3. Start the development server:
   npm run dev

## 🛠️ Built With

- React
- Vite
- Chakra UI
